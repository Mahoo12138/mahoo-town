# 红石与机器规则

服务器允许高频红石和自动化机器，但任何机器都不能影响服务器性能、公共区域稳定或其他玩家正常游玩。

## 允许范围

高频红石、自动农场、刷怪设施、物品分拣、机器仓库和其他生电工程都可以建造。服务器明确允许刷铁轨、刷地毯、TNT 复制机和打印机，但机器需要可控、可关闭，不能长期无意义空转。

| 项目 | 是否允许 | 要求 |
| --- | --- | --- |
| **刷铁轨** | 允许 | 不得造成大量掉落物堆积或影响 TPS。 |
| **刷地毯** | 允许 | 离开前尽量关闭，避免长期空转。 |
| **TNT 复制机** | 允许 | 仅用于合法工程，禁止破坏他人建筑、领地和公共设施。 |
| **打印机** | 允许 | 运行前确认范围、材料和性能影响，超大型打印工程需报备。 |

## 性能要求

| 项目 | 要求 |
| --- | --- |
| **红石时钟** | 离开前尽量关闭，不要长期空转。 |
| **实体数量** | 避免掉落物、生物、矿车、船等大量堆积。 |
| **漏斗与分拣** | 大型分拣系统建议先测试稳定性，再长期运行。 |
| **区块加载** | 不要让无人管理的机器持续影响多个区块。 |
